## 주요 기능

### 시트 병합
- 여러 엑셀 파일(.xlsx, .xlsm, .xltx, .xltm)을 하나의 파일로 병합
- **파일별 시트 선택** : 각 파일에서 원하는 시트만 선택하여 병합
- **공통 시트 선택** : 모든 파일에 공통으로 존재하는 시트를 일괄 선택
- 새 파일 생성 또는 기존 파일에 시트 추가(append) 모드 지원
- 시트명 충돌 시 자동으로 고유한 이름 생성

### 데이터 검증
병합 시 각 시트의 데이터를 자동으로 검증하고, 문제가 발견되면 **비고 열**에 기록합니다.

| 검증 유형 | 설명 |
|-----------|------|
| **빈값 탐지** | 비어있는 셀 검출 |
| **엑셀 오류값** | `#N/A`, `#VALUE!`, `#REF!`, `#DIV/0!` 등 오류 수식 검출 |
| **필수 입력** | 특정 열에 값이 반드시 있어야 하는 규칙 |
| **숫자만** | 숫자 값만 허용 (범위 지정 가능) |
| **전화번호** | 한국 전화번호 형식 검증 |
| **이메일** | 이메일 주소 형식 검증 |
| **날짜** | 날짜 형식(YYYY-MM-DD 등) 검증 |
| **정규식** | 사용자 지정 정규식 패턴 검증 |
| **허용값 목록** | 지정된 값만 허용 |

### 기타
- Windows 다크 모드 / 라이트 모드 자동 감지 및 테마 전환
- 실행 로그로 병합 진행 상황 실시간 확인
- 검증 규칙 JSON 파일로 저장/불러오기

## 사용 방법

### Step 1. 대상 파일 설정

프로그램 상단에서 병합 결과를 저장할 파일을 지정합니다.

- **기존 파일 선택** : 이미 존재하는 엑셀 파일에 시트를 추가합니다.
- **새 파일 만들기** : 새로운 엑셀 파일을 생성합니다.

### Step 2. 소스 파일 추가

병합할 엑셀 파일들을 추가합니다.

- `파일 추가` 버튼으로 하나 이상의 엑셀 파일을 선택
- `선택 제거`로 특정 파일 제거, `전체 제거`로 모두 제거
- 파일 목록에서 파일을 클릭하면 우측에 미리보기가 표시됩니다.

### Step 3. 시트 선택

각 파일에서 병합할 시트를 선택합니다.

- 파일 목록에서 파일을 선택하면 해당 파일의 시트가 체크박스로 표시됩니다.
- 기본적으로 모든 시트가 선택되어 있으며, 원하는 시트만 체크/해제할 수 있습니다.
- **공통 시트 선택** 버튼을 누르면 모든 파일에 공통으로 존재하는 시트를 팝업에서 선택하여 일괄 적용할 수 있습니다.

### Step 4. 읽기 옵션 설정

- **헤더 행** : 데이터의 헤더(열 제목)가 위치한 행 번호 (기본값: 1)
- **빈 행 건너뛰기** : 비어있는 행을 자동으로 제외

### Step 5. 검증 규칙 설정 (선택사항)

미리보기 영역에서 우클릭하여 열별 검증 규칙을 설정할 수 있습니다.

1. 미리보기 테이블의 열 헤더 또는 셀을 **우클릭**
2. 원하는 규칙 유형 선택 (필수 입력, 숫자만, 전화번호, 이메일, 날짜, 정규식, 허용값 목록 등)
3. 필요에 따라 세부 설정 (숫자 범위, 정규식 패턴, 허용값 등)

규칙은 파일별로 독립적으로 관리됩니다.

- **규칙 저장** : 설정한 규칙을 JSON 파일로 저장
- **규칙 불러오기 (이 파일)** : 현재 선택된 파일에만 규칙 적용
- **규칙 불러오기 (모든 파일)** : 추가된 모든 파일에 규칙 일괄 적용

### Step 6. 병합 실행

`병합 실행` 버튼을 클릭하면 병합이 시작됩니다.

- 좌측 하단의 실행 로그에서 진행 상황을 확인할 수 있습니다.
- 검증 문제가 발견되면 각 시트의 **비고** 열에 자동으로 기록됩니다.
- 완료 후 결과 파일을 열어 확인합니다.

## 지원 파일 형식

| 확장자 | 읽기 | 저장 |
|--------|------|------|
| `.xlsx` | O | O |
| `.xlsm` (매크로 포함) | O | O |
| `.xltx` (템플릿) | O | X |
| `.xltm` (매크로 템플릿) | O | X |
| `.xls` (구형) | X | X |
